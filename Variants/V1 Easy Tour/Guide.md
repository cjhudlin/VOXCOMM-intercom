General Guide

Options:
![ET_V1_Menu_p1](https://github.com/user-attachments/assets/8c4666e2-8db6-46f3-b45e-0bc87295b517)

- Device name - Max 9 Characters
- Voice assist - Informs the user what mode they are in
- MESH grp - Define here the name of the mesh group youd like to connect to
- MESH O / C - Choose the mesh type - Open or Closed. Choosing Open mesh option will allow any device using Vox mesh to communicate with you
- Duo mode - For use with 2 person communication
- MESH pass - Define here the password to be used for group communication mode
  
![ET_V1_Menu_p2](https://github.com/user-attachments/assets/045d911f-8b97-4230-b97c-0019c9999f85)

- Duo pass - Define here the password that will be used for 2 person communication
- Wifi Channel - Define here the channel that the Mesh group or duo mode will use
- Mic Gain - If users cannot here you well, try changing this option to raise the mic sensitivity
- BT Audio settings - This sub menu provides options for pre-defined sound modes (Equalizer)
- Audio device - a sub-menu containing information about the device, i.e. firmware version etc
  
![ET_V1_Menu_p3](https://github.com/user-attachments/assets/8bf44f1f-4f72-439e-a95f-953d5bc8fa4f)

- Help - a sub-menu containing various voiced help options
- Reset Defaults - Reset the device to default settings
