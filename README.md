# How to flash your device
1. Download the flashtool .zip from the variants folder.
2. Download the latest version of the firmware (.zip)
3. Extract the firmware to a folder
4. In the tool, choose the location of the firmware files you downloaded
5. In the tool, Choose the COM port the device is connected to
6. Click "Install firmware"!
