Thank you for showing interest in the Easy Tour V1 prototyping phase 1!

**Phase 1 is for maximum 100 participants**

There will be 3 phases - both phases are exactly the same, just spread out.
The prototyping Entry threshold period is 45 days starting as of 13 April 2026. (you may submit your request earlier)
- If before this date the maximum threshold is reached, the phase will start immediately after all necessary information has been collected and the first devices have been shipped.
- If after this date not enough participant show their interest, the phase will still start, it will just start at a later date
When the participation threshold has been met (and only then) would a request for the details of the participants (address, (contact number for shipping purposes only)) and payment request be sent to the participant

Cost: 80€ (not including shipping)
Requirement: EU only

This phase 1 for the Easy Tour V1 Prototype. Not only is this a chance for you to purchase a device at a cost saving, but also helpful for others that will use the device in the future
The phase duration is 3 months, during which time a number of questionairs may be sent to you. It is also helpful (and the whole purpose) to open issues, report bugs, and suggest new features / design changes during the time of the phase.

If you would like to participate in Phase 1:
1. Please read the PROTOTYPE_AGREEMENT.md file and follow the instructions.
2. In email, please provide the following:
   - Subject: Prototyping phase 1 - Participation request
   - Body: I [full name] would like to participate in Phase 1 of the Easy Tour V1 Prototype.

Thank you!
